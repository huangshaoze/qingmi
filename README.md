# qingmi-青咪(情迷)
Qingmi(青咪)是一个基于Flask二次开发的服务层框架， 技术栈采用了： python3, flask, Werkzeug, mongoengine, celery, fabric, httpie, Flask-Script, Flask-WTF, flask-mongoengine, Flask-Login, Flask-RESTful, Flask-DebugToolbar， Flask-Celery-Helper， requests, Flask-Caching, Flask-Admin, Flask-Uploads, httpie, fabric, ipython, Pillow等flask的第三方扩展库及工具。
